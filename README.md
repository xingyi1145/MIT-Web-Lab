# MIT Web Lab Course Projects

This repository contains my personal work and projects completed as part of the **MIT Web Lab** online course.  
The course focuses on web development fundamentals, including front-end and back-end technologies.

## Repository Structure

- **`main` branch** – Contains only this `README.md` file. It serves as an entry point and index for the repository.
- **Other branches** – Each branch corresponds to a specific project or lab exercise from the course.  
  Branch names are descriptive of the project content.

## How to Explore the Projects

1. Clone the repository:
   ```bash
   git clone https://github.com/xingyi1145/MIT-Web-Lab.git

2. List all available branches:
    ```bash
    git branch -r

3. Switch to a project branch to view its code and assets:
    ```bash
    git checkout <branch-name>

Each project branch includes its own README with specific details about that project.

## About MIT Web Lab
MIT Web Lab is an introductory web development course offered by MIT. It covers HTML, CSS, JavaScript, and server-side programming with Node.js and Express.
For more information, visit the official course site: [MIT Web Lab][https://site.weblab.is/].
